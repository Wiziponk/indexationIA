# Indexation IA

## Quickstart

```bash
# Local
git clone https://github.com/OWNER/indexationIA.git
cd indexationIA
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
flask --app app run --debug --port 5000

# Codespaces
python -m venv .venv && . .venv/bin/activate
pip install -r requirements.txt
flask --app app run --debug --port 5000
```
